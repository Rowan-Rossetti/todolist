# Focus — Todo list Vue 3

Une application de gestion de tâches moderne construite avec Vue 3, TypeScript et Vite.

## Fonctionnalités

- Création, modification, suppression et validation des tâches
- Description, priorité, catégorie, date d’échéance, tags et sous-tâches
- Tâches importantes et vues Aujourd’hui / À venir / Terminées
- Recherche instantanée, filtres et tris
- Statistiques et progression globale
- Sauvegarde automatique avec `localStorage`
- Mode clair et sombre
- Design responsive mobile, tablette et ordinateur
- Typographie Manrope + DM Sans

## Installation

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
